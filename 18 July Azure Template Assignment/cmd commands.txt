
1. Without parameters....

az deployment group create --name azurestorage --resource-group rgTemplate --template-file azuretemplateone.json

-------------------------------------------------------------------

2. By passing parameters during run....
az deployment group create --name azurestorage --resource-group rgTemplate --template-file azuretemplateone.json --parameters storageName=azurestoreg31910  storageAccountType=Standard_LRS


Command Breakdown: 

az deployment group create: The Azure CLI command to create a deployment at the resource group level.

--name: Specifies the name of the deployment (azurestorageUtpal).

--resource-group: Specifies the name of the resource group where the resources will be deployed (rgTemplateUtpal). 
If this resource group doesn't exist, you'll need to create it first with az group create.

--template-file: Specifies the path to the ARM template file (azureTemplateStorage.json).

--parameters: Specifies the template parameters. Here, storageName=myuniquestorage sets the storageName 
parameter to myuniquestorage.
